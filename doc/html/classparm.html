4.3 class parm

parm - Python extension for parsing and creating an SAP parameter to be added to an RFC Interface.

Generally you would not create one of these manually as it is far easier to use the "discovery" functionality of the conn.discover("RFCNAME") method. This returns a fully formed interface object. This is achieved by using standard RFCs supplied by SAP to look up the definition of an RFC interface and any associated structures. 

	import saprfc
	eparam = parm(name, structure, RFCEXPORT, datatype, intlen, decs, default, default)

parm(name="a name", structure="structure if complex", RFCEXPORT | RFCIMPORT, datatype="SAP datatype", intlen="internal parameter length", decs="decimal places", value="value", default="default value for reset")
creates a new RFC Interface Parameter object. This is either an import or export parameter (Export for parameters going into a call, Import for parameter values as a result of a call).
